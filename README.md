# Adventure-work
Sales and Profitability Analysis using Excel, SQL, and Tableau. Includes KPI calculations, data cleaning, and interactive dashboard.

## Tools Used:
- Power BI
- Excel
- SQL

## Project Description:
Analyzed sales data to identify yearly trends, top products, and regional performance.

## Key Insights:
- Highest sales in Q4
- West region generated maximum revenue
- Top 5 products contributed 40% revenue
